FUNKUHREN

Eine Funkuhr ist eine Uhr, die die Zeit über Radiosignale empfängt und automatisch die genaue Uhrzeit anzeigt. Das Signal wird von speziellen Sendern, die von nationalen Zeitdiensten betrieben werden, ausgestrahlt. Die Funkuhr hat eine Antenne, die das Signal empfängt und die Uhrzeit automatisch korrigiert, wenn nötig. Auf diese Weise wird sichergestellt, dass die Uhrzeit auf der Funkuhr immer genau und synchron mit der atomaren Zeit ist. Das macht die Funkuhr zu einer zuverlässigen Zeitmessung ohne manuelle Einstellung.
