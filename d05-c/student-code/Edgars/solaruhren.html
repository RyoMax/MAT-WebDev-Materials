SOLARUHR

Als künstliche Sonnenuhren gilt jede Vorrichtung, die es ermöglicht, die Zeit auf Basis der Sonnenposition zu messen. Eine Solaruhr ist eine spezielle Art von Sonnenuhr, die von einer solarbetriebenen Batterie betrieben wird, um die Zeit anzuzeigen. Die Solarzelle auf der Uhr fängt das Sonnenlicht ein und konvertiert es in elektrische Energie, die für den Betrieb der Uhr verwendet wird. Solaruhren sind umweltfreundlich und erfordern keine regelmäßigen Batteriewechsel. Sie eignen sich auch sehr gut für Outdoor-Aktivitäten, da sie nachts aufgeladen werden können und somit auch bei schwacher Beleuchtung eine zuverlässige Zeitmessung ermöglichen.
